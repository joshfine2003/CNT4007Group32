# CNT4007 Group 32
This project implements a P2P file sharing software similar to BitTorrent in Java.

## Requirements
- Java Development Kit (JDK)

## Compilation
```bash
cd src
javac peerProcess.java
```

## Sample Execution
```bash
cd src
java peerProcess 1001
java peerProcess 1002
java peerProcess 1003
```
